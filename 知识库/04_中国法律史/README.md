# 中国法律史

> 2025年国家统一法律职业资格考试大纲

## 考试归属

- **客观题试卷一**（理论法部分，占比较小）

## 基本要求

- **了解和理解**：中国历代主要法律思想与法律制度的基本特点、重要法典、司法制度
- **熟悉并能够运用**：中国法律发展演变的基本脉络，理解传统法律文化对当代法治建设的影响

## 考试内容

### 第一章 先秦时期的法律思想与制度

- **西周的法律思想与制度**
  - "明德慎罚"思想
  - 礼刑关系："出礼入刑"
  - 契约制度：质剂、傅别
  - 婚姻制度：一夫一妻制、同姓不婚、"七出三不去"
  - 继承制度：嫡长子继承制

- **春秋战国时期的法律思想与制度**
  - "铸刑书"（郑国子产）——中国历史上第一次公布成文法
  - "铸刑鼎"（晋国赵鞅）
  - 商鞅变法：改法为律
  - 法家思想：法、术、势

### 第二章 秦汉至魏晋南北朝时期的法律思想与制度

- **秦朝**
  - 法律形式：律、令、法律答问、封诊式、廷行事
  - 罪名与刑罚
  - 司法制度：廷尉

- **汉朝**
  - 法律思想：黄老思想到"德主刑辅"
  - 法律形式：律、令、科、比
  - 刑罚改革：文帝、景帝刑制改革（废除肉刑）
  - 春秋决狱
  - 录囚制度

- **魏晋南北朝**
  - 魏律（曹魏律）：改具律为刑名
  - 晋律（泰始律）：确立准五服以制罪
  - 北魏律：确立官当制度
  - 北齐律：首创十二篇体例，规定"重罪十条"
  - 法律儒家化进程

### 第三章 隋唐宋元时期的法律思想与制度

- **隋朝**
  - 《开皇律》：封建法典的代表
  - 确立封建制五刑：笞、杖、徒、流、死

- **唐朝**
  - 《唐律疏议》：中国现存最完整的封建法典
  - 唐律的基本精神：一准乎礼
  - 主要制度：十恶、六赃、六杀、保辜
  - 司法制度：大理寺、刑部、御史台
  - 唐律的历史地位：对东亚各国法律的影响

- **宋朝**
  - 法律形式：编敕、编例
  - 刑罚制度的变化：折杖法、刺配
  - 司法制度：审刑院、提点刑狱司
  - 契约制度的发展

- **元朝**
  - 法律特色：蒙汉异法
  - 司法制度的变化

### 第四章 明清时期的法律思想与制度

- **明朝**
  - 《大明律》：体例创新（名例律加六部分律）
  - 《大诰》：重典治吏
  - 刑罚制度的变化：廷杖
  - 司法制度：三法司（刑部、大理寺、都察院）
  - 厂卫干预司法

- **清朝**
  - 《大清律例》：中国历史上最后一部封建法典
  - 法律形式：律、例
  - 满汉异法
  - 司法制度
  - 清末修律：《大清新刑律》《大清民律草案》《大清商律草案》《法院编制法》
  - 清末司法制度改革：四级三审制

### 第五章 中华民国时期的法律思想与制度

- **南京临时政府**
  - 《中华民国临时约法》：中国历史上第一部资产阶级共和国宪法性文件
  - 司法改革措施

- **北洋政府**
  - 法律制度的特点
  - 《中华民国约法》（袁记约法）

- **南京国民政府**
  - "六法全书"体系：宪法、民法、刑法、民事诉讼法、刑事诉讼法、行政法
  - 法律制度的特点

### 第六章 新民主主义革命时期民主政权法制

- **工农民主政权时期**
  - 《中华苏维埃共和国宪法大纲》
  - 土地立法
  - 劳动立法
  - 婚姻立法
  - 刑事立法

- **抗日民主政权时期**
  - 施政纲领
  - 土地立法：减租减息
  - 劳动立法
  - 婚姻立法
  - 刑事立法
  - 司法制度：马锡五审判方式

- **解放战争时期**
  - 宪法性文件：《陕甘宁边区宪法原则》
  - 土地立法：《五四指示》《中国土地法大纲》
  - 刑事立法
  - 司法制度
