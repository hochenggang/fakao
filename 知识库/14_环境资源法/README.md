# 环境资源法

> 2025年国家统一法律职业资格考试大纲

## 考试归属

- **客观题试卷二**

## 基本要求

- **了解和理解**：环境保护法与自然资源法的基本制度
- **熟悉并能够运用**：环境保护的基本原则与制度、自然资源权属与管理

## 考试内容

### 第一章 环境保护法

#### 第一节 环境保护法概述
- 环境的概念与环境保护法的调整对象
- 环境保护法的基本原则：协调发展原则、预防为主原则、污染者负担原则、公众参与原则

#### 第二节 环境保护的基本制度
- 环境影响评价制度
- "三同时"制度（同时设计、同时施工、同时投产使用）
- 排污许可制度
- 环境标准制度
- 环境监测制度
- 环境信息公开制度
- 生态保护补偿制度

#### 第三节 环境法律责任
- 环境行政责任
- 环境民事责任：无过错责任原则、因果关系推定、举证责任倒置
- 环境公益诉讼：原告资格、程序规则
- 环境刑事责任

### 第二章 自然资源法

#### 第一节 自然资源法概述
- 自然资源的概念与分类
- 自然资源权属制度：国家所有、集体所有
- 自然资源利用管理制度

#### 第二节 森林法
- 森林资源的权属
- 森林采伐制度
- 森林保护制度

#### 第三节 草原法
- 草原权属制度
- 草原保护建设利用

#### 第四节 矿产资源法
- 矿产资源的权属（国家所有）
- 探矿权与采矿权
- 矿产资源的勘查与开采管理

#### 第五节 水法
- 水资源的权属（国家所有）
- 取水许可制度
- 水资源保护

#### 第六节 渔业法
- 渔业权
- 养殖业与捕捞业管理
- 渔业资源保护

#### 第七节 野生动物保护法
- 野生动物保护制度
- 野生动物猎捕和驯养繁殖管理
