# 经济法

> 2025年国家统一法律职业资格考试大纲

## 考试归属

- **客观题试卷二**

## 基本要求

- **了解和理解**：竞争法、消费者法、银行业法、财税法、土地法和房地产法的基本制度
- **熟悉并能够运用**：反垄断法、反不正当竞争法、消费者权益保护法等，分析经济法案例

## 考试内容

### 第一章 竞争法

#### 第一节 反垄断法
- 反垄断法概述：立法目的、适用范围
- **垄断协议**：横向垄断协议（固定价格、限制产量、分割市场、限制新技术、联合抵制）、纵向垄断协议（固定转售价格、限定最低转售价格）、垄断协议的豁免
- **滥用市场支配地位**：不公平定价、掠夺性定价、拒绝交易、限定交易、搭售、差别待遇
- **经营者集中**：申报标准、审查程序、审查标准、附加限制性条件
- **滥用行政权力排除限制竞争**：行政垄断的表现形式
- 反垄断法的实施：执法机构、法律责任

#### 第二节 反不正当竞争法
- 反不正当竞争法概述
- **不正当竞争行为**：
  - 混淆行为：假冒他人注册商标、擅自使用知名商品特有的名称包装装潢、擅自使用他人企业名称、虚假表示
  - 商业贿赂
  - 虚假宣传
  - 侵犯商业秘密
  - 不正当有奖销售
  - 商业诋毁
  - 网络不正当竞争行为
- 法律责任

### 第二章 消费者法

#### 第一节 消费者权益保护法
- 消费者的概念与特征
- **消费者的权利**：安全保障权、知情权、选择权、公平交易权、获得赔偿权、结社权、获得知识权、人格尊严权、监督权
- **经营者的义务**：保障安全义务、真实信息告知义务、出具凭证义务、质量保证义务、售后服务义务、不得以格式条款排除消费者权利
- 消费者权益的保护：消费者协会、争议解决途径
- 法律责任：民事责任（惩罚性赔偿）、行政责任

#### 第二节 产品质量法
- 产品与产品质量的概念
- 产品质量监督管理：标准化管理、质量认证制度、产品质量监督检查
- 生产者销售者的产品质量义务
- 产品责任：生产者的严格责任、销售者的过错责任、缺陷产品的召回

#### 第三节 食品安全法
- 食品安全法概述
- 食品安全风险监测与评估
- 食品安全标准
- 食品生产经营
- 食品检验
- 食品安全事故处置
- 法律责任

### 第三章 银行业法

#### 第一节 商业银行法
- 商业银行的概念与特征
- 商业银行的设立与组织
- 商业银行的业务：负债业务、资产业务、中间业务
- 商业银行的管理
- 商业银行的接管与终止

#### 第二节 银行业监督管理法
- 银行业监督管理机构
- 监督管理职责
- 监督管理措施
- 法律责任

### 第四章 财税法

#### 第一节 税法
- 税法的概念与体系
- 税法的基本原则：税收法定原则、税收公平原则、税收效率原则
- 税法的构成要素：纳税人、征税对象、税率、纳税环节、纳税期限、税收优惠
- **增值税法**：纳税义务人、征税范围、税率与征收率、应纳税额计算
- **企业所得税法**：纳税义务人、应纳税所得额、税率、税收优惠
- **个人所得税法**：纳税义务人、应税所得项目、税率、专项附加扣除
- 税收征收管理法：税务管理、税款征收、税务检查、法律责任

#### 第二节 审计法
- 审计法概述
- 审计机关和审计人员
- 审计职责
- 审计权限
- 审计程序

### 第五章 土地法和房地产法

#### 第一节 土地管理法
- 土地所有权与使用权：国家所有、集体所有
- 土地用途管制制度：农用地、建设用地、未利用地
- 耕地保护制度
- 建设用地制度：农用地转用审批、土地征收
- 国有建设用地使用权：出让、划拨
- 集体建设用地使用权
- 土地监督检查

#### 第二节 城乡规划法
- 城乡规划的概念与体系
- 城乡规划的制定
- 城乡规划的实施

#### 第三节 城市房地产管理法
- 房地产开发用地
- 房地产开发
- 房地产交易：转让、抵押、租赁
- 房地产权属登记

#### 第四节 不动产登记暂行条例
- 不动产登记的概念与类型
- 不动产登记的程序
- 不动产登记的效力
