# 法理学

## 一、法的本体

### 1.1 法的概念
- 法的定义：由国家制定或认可，以权利义务为内容，由国家强制力保障实施的社会规范
- 法的特征：规范性、国家意志性、普遍性、强制性、程序性

### 1.2 法的作用
- 规范作用：指引、评价、预测、教育、强制
- 社会作用：政治职能、社会职能

### 1.3 法的价值
- 秩序、自由、正义、效率、人权
- 价值冲突及其解决原则
