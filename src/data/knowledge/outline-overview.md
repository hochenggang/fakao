# 法考大纲总览

## 考试结构

国家统一法律职业资格考试分为两个阶段：

### 客观题考试
- **试卷一（公法卷）**：上午 9:00-12:00，150分
  - 习近平法治思想、法理学、宪法、中国法律史、国际法
  - 司法制度和法律职业道德、刑法、刑事诉讼法、行政法与行政诉讼法

- **试卷二（私法卷）**：下午 14:30-17:30，150分
  - 民法、知识产权法、商法、经济法、环境资源法
  - 劳动与社会保障法、国际私法、国际经济法、民事诉讼法（含仲裁制度）

- **合格线**：两卷合计180分

### 主观题考试
- **考试时间**：上午 9:00-13:00，4小时，180分
- **考试形式**：计算机化考试，键盘输入作答
- **合格线**：108分
- **考查科目**：法治思想、法理学、宪法、刑法、刑事诉讼法、民法、商法、民事诉讼法、行政法与行政诉讼法、司法制度和法律职业道德

## 备考建议

1. **先修理解，后记忆** — 法律重在理解原理，死记硬背效果差
2. **真题为王** — 反复研习历年真题，把握命题规律
3. **重者恒重** — 民法、刑法、民事诉讼法、刑事诉讼法是重中之重
4. **注重体系** — 建立各科知识体系，融会贯通
5. **主观题早准备** — 不要等客观题通过后再准备主观题
