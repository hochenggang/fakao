# 刑法

## 一、犯罪构成

### 1.1 四要件体系
- 犯罪客体
- 犯罪客观方面
- 犯罪主体
- 犯罪主观方面

### 1.2 犯罪客体
- 一般客体、同类客体、直接客体

### 1.3 犯罪客观方面
- 危害行为、危害结果、因果关系

### 1.4 犯罪主体
- 自然人：刑事责任年龄、刑事责任能力
- 单位犯罪

### 1.5 犯罪主观方面
- 故意：直接故意、间接故意
- 过失：疏忽大意、过于自信
