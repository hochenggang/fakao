# 知识产权法

## 一、著作权

### 1.1 作品
- 构成要件
- 不受保护的对象

### 1.2 权利内容
- 人身权：发表权、署名权、修改权、保护作品完整权
- 财产权：复制权、发行权、出租权、展览权、表演权、放映权、广播权、信息网络传播权等

### 1.3 保护期限
- 人身权：永久（发表权除外）
- 财产权：作者终生加50年

## 二、专利权

### 2.1 授予条件
- 新颖性、创造性、实用性

### 2.2 权利内容
- 发明、实用新型、外观设计

### 2.3 侵权判定
- 全面覆盖原则

## 三、商标权

### 3.1 注册条件
- 显著性
- 非功能性
- 不违反禁用条款

### 3.2 权利内容
- 专用权、禁止权

### 3.3 侵权判定
- 混淆可能性
